## Usage

Used in the root `README.md` header. The mark is a stylised **M** set inside a globe crossed by a
vertical meridian line — the line the project is named for — with a reticle framing it.

The cyan accent matches the C0 interface accent token (`--accent` in
`scaffold/frontend/src/tokens.css`), so the mark and the product surface read as one system. If
either changes, change both.
